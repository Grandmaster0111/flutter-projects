# Flutter Projects

A collection of Flutter UI experiments and mini-apps.

## Projects

### Brightness Adjuster (`flutter app to adjust brightness/`)
A Flutter app that controls the device screen brightness using a slider UI.

- Uses the `screen_brightness` plugin
- Real-time brightness adjustment with a clean slider interface
- Works on Android and iOS

#### Run

```bash
cd "flutter app to adjust brightness"
flutter pub get
flutter run
```

## Prerequisites

- Flutter SDK 3.x — [flutter.dev](https://flutter.dev/docs/get-started/install)
- Android Studio or Xcode for device/emulator
